# GameProject - Hero's Path

## Краткое описание

Пошаговая RPG в консоли, разрабатываемая в рамках 3 курса МПГУ, факультета ИМО по предмету "ООП и Гибкая методология разработки". 

## Концепция

Игрок исследует подземелья, сражается с врагами и прокачивает персонажа. 
Проект создан для изучения Git и паттернов проектирования.
